# imgs_789_skin_representation_and_ood
This project investigates how few-shot learning and out-of-distribution generalization can improve skin appearance modeling under real-world dataset bias and variation in lighting, facial region, and identity. While my computer vision project in IMGS 621 focuses on developing more stable and informative multi-region representations of skin tone, this work builds on that foundation by studying how those representations behave under distribution shift and limited data, and how models can be made more robust through similarity learning and OOD-aware techniques. Together, these projects aim to separate the challenges of representation and generalization in order to move toward more reliable cosmetic shade mapping systems.
